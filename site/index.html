Native macOS workspace for Hermes Agent

Stay close to the host.

Hermes Desktop - OS1 Edition is a native Mac app for Hermes Agent. It brings sessions, files, skills, cron jobs, Kanban dashboard, usage stats and a real terminal into one calm place, with the host as the source of truth.

Included

Open the app, choose an SSH profile, and work against the same remote Hermes state you already use: sessions, Kanban, files, usage, skills, cron jobs, and shell.

Connections

Connect to your fleet

Save direct SSH profiles for the default Hermes home or named Hermes profiles on the same host.

Sessions

Pick up any thread

Browse history, search transcripts, pin important work, continue a chat, or resume the session in Terminal.

Kanban

Orchestrate your agents

Create, assign, comment, block, complete, and inspect tasks from the host-wide ~/.hermes/kanban.db.

Files + skills

Edit the live workspace

Update canonical Hermes files, bookmarked remote text files, and SKILL.md files with conflict checks before save.

Usage + cron

Review usage and schedules

Review usage totals, model breakdowns, trends, and manage recurring cron jobs from the canonical scheduler state.

Terminal

Open a real SSH terminal

Run a real embedded SSH terminal beside the rest of the workspace, and switch tabs across profiles.

Positioning

Browser for administration, Mac app for host work. Hermes Desktop - OS1 Edition gives the existing SSH path a native surface.

Before you download

Simple setup.

If SSH already works from this Mac to your Hermes host, setup is usually straightforward.

ssh your-host
  • macOS 14 or newer
  • python3 available on the Hermes host
  • hermes available on the host’s non-interactive SSH path
  • Hermes data under the remote user’s ~/.hermes
Trust model

Easy to understand.

No helper service on the host. No gateway API. No separate desktop database pretending to be the source of truth.
